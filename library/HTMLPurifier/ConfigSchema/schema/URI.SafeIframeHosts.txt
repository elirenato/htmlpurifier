URI.SafeIframeHosts
TYPE: lookup/null
DEFAULT: null
--DESCRIPTION--
<p>
    A whitelist which indicates what explicit hosts should be
    allowed to embed iframe. Here are some example values:
</p>
<ul>
    <li><code>www.youtube.com</code> - Allow YouTube videos</li>
    <li><code>maps.google.com</code> - Allow Embedding a Google map</li>
</ul>
<p>
    Note that this directive works in conjunction with %HTML.SafeIframeRegexp.
</p>
